# 🏋️‍♂️ Fitness Tracker Console App

A **Java-based console application** designed to help users track their fitness activities, manage workout plans, set goals, and get AI-powered fitness suggestions.

---

## 🚀 Features

* 🔐 **User Authentication**

  * Sign Up & Login functionality
* 🏃 **Activity Tracking**

  * Record activities (Running, Walking, Cycling, etc.)
  * Track duration, distance, calories burned, pace, and heart rate
* 📋 **Workout Plans**

  * Create and view personalized workout plans
* 🎯 **Fitness Goals**

  * Set and monitor fitness goals (weight, distance, etc.)
* 🤖 **AI Fitness Advisor**

  * Get smart workout suggestions based on user input
* 📊 **Console-Based UI**

  * Clean tabular display for activities and plans

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```
   git clone https://github.com/your-username/FitnessTracker.git
   ```

2. Open the project in your IDE.

3. Configure MySQL:

   * Create required tables (`user`, `activity`, `workout`, `fitness_goal`)
   * Update database credentials in your code

4. Run the application:

   ```
   Run FintnessApp.java
   ```
